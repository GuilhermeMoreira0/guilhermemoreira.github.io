Projeto Jari

Jari é um projeto de inscrição para membros do Jari/CET.

Eu, junto com mais 2 pessoas, fizemos o Jari usando o Visual Studio 2022, C# .NET Core MVC, HTML, CSS, Javascript, SQL Server.

Também desenvolvemos a página de Gestão para controle de inscrições. que tem o login, e uma parte de relatorios
