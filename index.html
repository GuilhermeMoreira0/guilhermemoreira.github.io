MEU PORTFÓLIO

JARI

O projeto "JARI" foi desenvolvido em C# e .NET Core como uma ficha de inscrição. Foi o meu primeiro projeto realizado em equipe, com o front-end construído em HTML, CSS e JavaScript e o back-end em C#. Apesar das dificuldades iniciais com validações nos campos devido à inexperiência, o apoio da equipe foi essencial para superá-las. Além disso, foi implementado um sistema de gestão que restringe o acesso aos dados das fichas de inscrição apenas a pessoas específicas.

Espaider

O projeto "Espaider" foi desenvolvido em C# e .NET Core para automatizar um serviço anteriormente manual, utilizando HTML, CSS, JavaScript, SQL Server e Bootstrap para o front-end. A biblioteca PPlus foi integrada para manipulação de arquivos Excel. Relatórios foram implementados no ReportService após migrar seis relatórios do Crystal Reports, superando desafios técnicos por meio de aprendizado autodirigido. Desenvolvido inteiramente no Visual Studio 2022, o projeto reflete inovação e adaptação.

Ouvidoria

O projeto "Ouvidoria" consistiu na realização de diversas alterações em um site existente desenvolvido em C# ASP.NET. A aplicação, voltada para a inscrição de requerimentos, inclui validação de protocolos que direciona para outra página. Embora as alterações fossem pequenas, o desconhecimento inicial da tecnologia tornou o início desafiador. Após estudar o código, Eu conseguiu implementar as mudanças com sucesso.
